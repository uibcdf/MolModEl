# MolModEl

```
atom ---> group ---------> entity -----> bioassembly
            |                |               |
             ---> segment --->               |         # in case of polymers
                             |               |
                              ---> chain --->          # in case of coming from pdb
```




```
entity -------------> ion
             |
              ------> water
             |
              ------> small molecule
             |
              ------> peptide
             |
              ------> protein
```

```
bioassembly ------> protein-protein complex
             |
              ------> protein-ligand complex
             |
              ------> protein-DNA complex
             |
              ------> protein-ion complex
             |
              ------> any biomolecular association
```
